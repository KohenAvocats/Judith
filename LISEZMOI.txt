CONTENU DU ZIP
- decisions/      41 955 decisions de justice rendues en avril 2026
                  (Cour de cassation, Cours d'appel, Tribunaux judiciaires,
                  Tribunaux de commerce) au format Markdown.
- import.py       Script Python qui se charge de tout.
- Importer-dans-Notion.command   Lanceur Mac (double-clic).
- Importer-dans-Notion.bat       Lanceur Windows (double-clic).

UTILISATION (Mac)
1. Decompressez le ZIP n'importe ou (Bureau, Telechargements, etc.).
2. Double-cliquez sur "Importer-dans-Notion.command".
   - Si Mac refuse : clic droit -> Ouvrir -> Ouvrir (1ere fois seulement).
3. Une fenetre Terminal s'ouvre, puis votre navigateur ouvre Notion.
4. Connectez-vous a Notion, cliquez "Selectionner les pages",
   choisissez une page parente, cliquez "Allow".
5. De retour dans le Terminal, choisissez le numero de la page parente.
6. Laissez tourner ~3 a 4 heures. Vous pouvez fermer et relancer
   plus tard : l'import reprendra ou il en etait.

UTILISATION (Windows)
1. Decompressez le ZIP.
2. Double-cliquez sur "Importer-dans-Notion.bat".
3. Suivez les memes etapes que ci-dessus.

PRE-REQUIS
- Python 3 installe (https://www.python.org/downloads/).
  Sur Mac, generalement deja present.
  Sur Windows, cochez "Add Python to PATH" pendant l'installation.

Cet outil ne fournit aucun conseil juridique et ne remplace pas

EN CAS DE PROBLEME

- "Aucune page partagee" : dans Notion, ouvrez une page, cliquez "..."
