## Version 1 — Sobre, expert, public avocats

41 955 décisions de justice rendues en avril 2026 (Cour de cassation, cours d'appel, tribunaux judiciaires, tribunaux de commerce), importables dans votre Notion en deux clics, gratuitement.

J'ai mis le tout dans un dépôt GitHub open source : un script vous authentifie sur votre compte Notion, crée une base de données dans un workspace de votre choix, et y pousse les 41 955 décisions avec leurs métadonnées (juridiction, date, chambre, numéro de pourvoi). Ensuite, l'IA native de Notion peut interroger ce corpus en langage naturel : « Décisions sur la garde alternée d'un animal de compagnie », « Cyberharcèlement scolaire en avril 2026 », « Litiges en matière de cryptomonnaies devant les tribunaux français », etc.

L'idée : permettre à chaque confrère de constituer chez lui un premier corpus jurisprudentiel exploitable par une IA, sans abonnement à une plateforme, sans clé API à gérer, sans configuration technique. Le code est ouvert, les données proviennent de l'API officielle Judilibre sous Licence Ouverte Etalab 2.0.

Pour l'installation détaillée et le téléchargement :
https://github.com/KohenAvocats/jurisprudence-vers-notion

Disclaimer technique. Une fois importées dans votre workspace, vous devenez responsable du traitement RGPD des données. Cet outil ne fournit aucun conseil juridique et ne dispense pas d'une vérification sur les sources officielles avant tout usage professionnel.

## Conseils de publication

**Format** : poster la vidéo en upload natif LinkedIn (pas en lien YouTube). LinkedIn favorise les vidéos natives dans son algorithme.

**Durée optimale vidéo** : entre 60 et 90 secondes. Si la tienne fait plus, considère couper le début/fin.

**Heure de publication** : mardi ou jeudi entre 8h-9h ou 17h-18h pour le public juridique français.

**Hashtags** : 3 à 5 max, à la fin. Suggestions : #LegalTech #IA #Jurisprudence #Avocat #Notion

**Premier commentaire à publier toi-même** : le lien GitHub. LinkedIn pénalise les liens dans le post principal mais pas dans les commentaires.

**Ma version recommandée** : la **Version 3**. Plus courte, accroche directe, structure LinkedIn-friendly.
